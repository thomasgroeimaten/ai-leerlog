## Geselecteerde video

### 1. IndyDevDan — "Building Multi-Agent Systems With Finite State Machines // Applied AI Meetups July 2026"
- **URL:** https://www.youtube.com/watch?v=Onq9nHMN0oQ
- **Gepubliceerd:** juli 2026 (Applied AI Meetups-serie)
- **Kernboodschap:** FSM (Finite State Machine) en Actor Model als architectuurparadigma voor deterministisch beheersbare multi-agent systemen. "De wereld is state machines all the way down."
- **Ruwe notities:**
  - FSM = wiskundig model dat definieert welke agent verantwoordelijk is per fase + exacte transitiecondities naar de volgende staat
  - Actor Model: elke agent verwerkt berichten asynchroon, zonder gedeelde mutable state → schaalbaar en betrouwbaar
  - Combinatie FSM + Actor Model = voorspelbare, auditeerbare orchestratie van parallelle agent-stromen
  - 2026 = jaar van multi-agent systemen — infrastructuur is nu volwassen genoeg voor productie
  - MetaAgent-framework (gerelateerd academisch werk) bewijst dat FSM-gebaseerde multi-agent systemen automatisch construeerbaar zijn vanuit een taakomschrijving
  - Historische context: FSM en Actor Model zijn bewezen patronen uit distributed systems — geen nieuwe hype, maar hergebruik van solide engineering
