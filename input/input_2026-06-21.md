### 1. Cole Medin — "The Creators of Claude Code and OpenClaw don't Prompt Their Agents Anymore?!"
- **Kanaal:** Cole Medin
- **Gepubliceerd:** 2026-06-18
- **URL:** https://www.youtube.com/watch?v=UztrFXaSWv0
- **Ruwe notities:**
  - De beste agentic engineers tikken geen losse prompts meer — ze bouwen loops die voor hen prompten en laten die uren draaien
  - Boris Cherny (Claude Code-creator) + OpenClaw-creator: werken harness-first, niet prompt-first
  - OpenClaw-patroon: `SOUL.md` (karakter + persistente context) + `user.md` (wie ben ik / wat wil ik) — voor iedere sessie geïnjecteerd
  - Als je jezelf betrapt op het herschrijven van dezelfde prompt → signaal om een loop te bouwen

### 2. Nate Herk — "How to Build Effective Claude Code Agents in 2026"
- **Kanaal:** Nate Herk | AI Automation
- **Gepubliceerd:** 2026-06-18
- **URL:** https://www.youtube.com/watch?v=RzLV8sfFdMM
- **Ruwe notities:**
  - Planning + verificatie scheidt "vibe coding" van echte resultaten
  - Elk model heeft een "dumb zone": een punt waarop het evident eenvoudige dingen begint te missen (context-drift)

  - Plan-fase eerst: laat agent een plan schrijven, geef goedkeuring, pas dan executie
