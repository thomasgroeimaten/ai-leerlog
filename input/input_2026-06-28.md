### Cole Medin — "Google Just Dropped a Masterclass on Agentic Engineering"
Google 50-pagina's playbook: "vibe coding doesn't scale"
Cole's workshop GitHub (ai-transformation-workshop):
- AI Layer concept: tweede codebase naast productiecode
  - CLAUDE.md = altijd geladen (tech stack, patronen, standaarden)
  - On-demand context = referentiedocs + stijlgidsen
  - Commands/skills = gestandaardiseerde teamworkflows
- PIV Loop: Plan → Implement → Validate
  - Plan: context laden + gestructureerd implementatieplan
  - Implement: uitvoeren in vers contextvenster
  - Validate: vijflaags piramide (typecheck → unit → integratie → e2e → handmatig)
- 5 Golden Rules:
  1. Herhaalbare taken → herbruikbare commands
  2. Vragen boven aannames
  3. Context strategisch resetten tussen grote fasen
  4. Beschrijvende git-history als agentgeheugen
  5. Agentfouten = kansen om AI-laag te verbeteren
- 15 commands: /prime, /create-prd, /create-stories, /plan, /implement, /validate, /review + varianten
Bronnen:
